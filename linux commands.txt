pwd - print working(current) directory

ls - list files and directories
ls -l long
ls -a all
ls -R: recursively list directory contents
ls -r: reverse order while sorting
ls -t: sort by modification time, newest first
   -h: human-readable, used with -l or -s, print sizes like 1K 234M 2G etc.
   -1: one file per line
example: ls -la, ls -latr, ls /home/pete

cd - change directory
. (current directory). This is the directory you are currently in.
.. (parent directory). Takes you to the directory above your current.
~ (home directory). This directory defaults to your “home directory”. Such as /home/pete.
- (previous directory). This will take you to the previous directory you were just at.
cd without flags takes you to home directory

date

whoami

touch <filename> - create new file or change timestamps on existing files and directories
touch myfile, touch application.properties

clear

mkdir - make directory(ies)
mkdir myfolder
mkdir -p Linux/Debian/Pete

rmdir - remove empty directory(ies)
rmdir Linux/ - results in error: rmdir: failed to remove 'Linux/': Directory not empty
rmdir -p Linux/Debian/Pete




Set an alias in your ~/.bash_profile file. / .bashrc
alias ls="ls -lah"
alias ll="ls -lh --color"
alias l="ls -1"

which <executable> - display full path to a command's executable file. 
'which' can only locate executables, not shell built-in commands.
operates by searching through the directories listed in the PATH environment variable
which ls
which java
